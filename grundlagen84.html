Russische Grammatik: Präposition ПО

Kurzanleitung

Die Präposition ПО gibt an, dass jeder „je“ eine bestimmte Menge erhält:

  • Zahl 1: Dativ + Dativ (по одному билету)
  • Zahl 2, 3, 4: Akkusativ (Zahl) + Genitiv Singular (по два билета)
  • Zahl 5–20: Akkusativ (Zahl) + Genitiv Plural (по пять билетов)

Hinweis: Alle Aufgaben geben sofort Feedback. Grün = Richtig, Rot = Falsch.
